Calculates the Cognitive Complexity of each non-abstract method. The metric is similar to Cyclomatic Complexity, but is intended to explicitly measure understandability, which can be quite different from testability. Cognitive Complexity is increased with each control structure used and is higher the more nested control structures are.

For more information on the Cognitive Complexity metric read the White Paper.

Abbreviation: CogC
