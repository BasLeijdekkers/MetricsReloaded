Calculates the degree of coupling in a project as a whole. Essentially it reports what proportion of the classes in a project are used by (couple to) an average class in the project.

For more information on the Coupling Factor and other MOOD metrics read the article The Design of Eiffel Programs: Quantitative Evaluation Using the MOOD Metrics by Fernando Brito e Abreu.
