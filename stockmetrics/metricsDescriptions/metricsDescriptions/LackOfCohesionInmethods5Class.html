Calculates the degree of cohesiveness of a class. This is on—É of the variants of Lack of cohesion in methods metric. For each field it finds a set of methods from the same class that use this field. Value of the metric is the mean of the Jaccard distance between two sets of methods for each pair of fields. For more information see Henderson-Sellers, Constantine and Graham. Coupling and Cohesion (Towards a Valid Metrics Suite for Object-Oriented Analysis and Design). Object-Oriented Systems, 3(3), pp 143-158, 1996
