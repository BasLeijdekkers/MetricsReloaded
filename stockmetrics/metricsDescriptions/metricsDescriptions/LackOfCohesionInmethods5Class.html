Calculates on the degree of cohesiveness of a class. This is on of the variants of Lack of cohesion in methods metric. For each field we find set of methods from the same class, which uses this field. Value of the metric is the mean of the Jaccard distance between two sets of methods for each pair of fields. For more information see Gabriela Czibula, Zsuzsanna Marian, Istvan Gergely Czibula (2015) Detecting software design defects using relational association rule mining
