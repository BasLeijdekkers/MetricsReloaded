Calculates the degree of cohesiveness of a class. This is one of the variants of Lack of cohesion in methods metric. The metric is the difference between number of method pairs that do not access common attributes and method pairs that do. If that is negative number, then LCOM1 is set to 0. For more information see S.R.Chidamber, C.F.Kemerer. A Metrics Suite for Object Oriented Design. IEEE Transactions on Software Engineering, Vol. 20, No. 6, June 1994
Abbreviation: LCOM1
