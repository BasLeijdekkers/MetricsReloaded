Calculates the number of efferent couplings for each package. An efferent coupling is a reference from a class or interface internal to the package to a class or interface external to the package. Test classes and library classes are not included. For more information on the efferent coupling metric and how it is used, see http://www.objectmentor.com/resources/articles/oodmetrc.pdf.
